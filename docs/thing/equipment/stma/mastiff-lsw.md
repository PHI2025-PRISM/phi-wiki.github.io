# **“獒犬”轻型支援武器系统（Mastiff-LSW）**

## **核心设计理念**
专为高威胁机动突击作战设计的轻量化压制武器，在7.5kg重量级实现每分钟1000发的高射速火力，同时保持单兵携行机动性。

## **技术参数**
- **全枪长**：950mm（枪托展开）/720mm（枪托折叠）
- **枪管长**：415mm（带速换机构）
- **空枪重**：7.2kg（含两脚架）
- **战斗全重**：9.8kg（含100发弹链箱）
- **理论射速**：1000rpm（可调至850rpm）
- **供弹方式**：100发半透明聚合物弹链箱/50发快装弹鼓
- **冷却系统**：强制气冷+可更换枪管（3秒热更换）

## **核心创新系统**

### **1. 复合后坐控制技术**
- **双路径缓冲系统**：枪机后坐与枪管短后坐分离，60%后坐能量通过液压缓冲器在枪托内耗散
- **主动枪口制退器**：三腔室设计+顶部泄气孔，降低枪口上跳40%
- **平衡块往复系统**：枪机组下方配置230g反向运动配重块，抵消70%可感后坐力

### **2. 高效散热结构**
- **枪管**：镀铬硬化钢，带螺旋散热肋片，增加散热面积300%
- **导气管路**：钛合金耐热导气箍，内置热交换鳍片
- **智能过热保护**：枪管护木内嵌温度传感器，超温时扳机阻力增大预警

### **3. 弹药系统**
- **弹药规格**：7.62×45mm APEX（穿甲扩展型）
  - 弹头结构：碳化钨穿甲芯（直径5.8mm）+ 铝合金风帽 + 铜锌被甲
  - 装药：双基球形药，膛压峰值380MPa
  - 弹壳：覆铜钢瓶颈弹壳，带环形沟槽防抱膛
- **性能数据**：
  - 初速：860m/s（415mm枪管）
  - 100m穿深：击穿18mm Rolled Homogeneous Armor（相当于NIJ IV级）
  - 有效射程：800m（点目标）/1200m（压制射击）
